# 🐢 Random Turtle Walk (game.py)

## 📌 Overview

A simple Python program using the `turtle` module to draw a random colorful path.

## ⚙️ Requirements

* 🐍 Python 3.x

## ▶️ Run

```bash id="2k9lqm"
python game.py
```

## 🎨 What it does

* 🎲 Moves the turtle in random directions
* 🌈 Changes color at each step
* 🔁 Repeats 200 times
* 🖱️ Click the window to exit

## 📝 Notes

* 🔢 Change steps in `range(200)`
* ⏱️ Adjust speed using `time.sleep()`
